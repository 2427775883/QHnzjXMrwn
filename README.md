# Python车主之家数据分析系统简介

## 核心功能
- 登录注册
- 汽车历史销量数据爬取
- 汽车总体月销量分析
- 新能源汽车品牌月度销量预测
- 汽车总体年销量分析
- 各汽车品牌年度总销量对比分析

## 适用场景
- 数据分析行业
- 车辆销售市场研究
- 汽车行业决策支持

## 技术特点
- 使用Flask框架搭建Web应用
- 采用Bootstrap进行页面设计
- 利用Echarts进行数据可视化展示
- 基于mysql数据库存储数据
- 使用HTML、CSS、JavaScript进行前端开发
- 集成ARIMA、决策树回归、Ridge岭回归等预测算法
